# German Learning Curriculum - Level B1

Curriculum based on CEFR guidelines and Goethe-Institut for B1 level (Threshold/Intermediate).

## B1 Level Description

A person at B1 level (according to official CEFR guidelines):
- Understands the main points of clear standard input on familiar matters regularly encountered in work, school, leisure
- Can deal with most situations likely to arise while traveling in an area where the language is spoken
- Can produce simple connected text on topics that are familiar or of personal interest
- Can describe experiences and events, dreams, hopes and ambitions
- Can briefly give reasons and explanations for opinions and plans

B1 is the "threshold" level - the level of language independence.

## B1 Grammar

### Verbs with Prepositions
- sich freuen auf/über, sich interessieren für
- warten auf, sich erinnern an
- träumen von, denken an
- Fixed prepositional phrases

### Temporal Clauses
- als (one-time past events)
- wenn (repeated or future)
- nachdem (time sequence)
- bevor, während, bis

### Causal and Purpose Clauses
- weil, da (cause)
- damit (purpose with different subjects)
- um...zu (purpose with same subject)
- deshalb, deswegen (consequence)

### Concessive Clauses
- obwohl (although)
- trotzdem (nevertheless)
- trotz + Genitive (despite)

### Konjunktiv II
- würde + Infinitive (periphrastic form)
- Synthetic forms (wäre, hätte, könnte)
- Polite requests (Könnten Sie...)
- Wishes and advice (An deiner Stelle würde ich...)

### Passive Voice (Passiv)
- Vorgangspassiv (werden + Partizip II)
- Modal verbs with Passive
- Passive in different tenses

### Relative Clauses (Relativsätze)
- Relative pronouns (der, die, das, welcher)
- Relative clauses with prepositions
- was as relative pronoun

### Adjective Declension
- After definite article
- After indefinite article
- Without article

### Genitive (Genitiv)
- Prepositions with genitive (wegen, trotz, während)
- Noun declension in genitive
- Use in formal style

### Future Tense (Futur I)
- werden + Infinitive
- Present tense with time indicator
- Plans and predictions

### Partizip II as Adjective
- das geöffnete Fenster
- der geschriebene Brief
- Use in descriptions

## Grammar Mapping to Topics

Grammar is integrated with topics - each set contains phrases using specific grammatical constructions.

| Topic | Grammar Constructions to Use |
|-------|------------------------------|
| **Travel - planning and reservations** | Konjunktiv II (Ich möchte, Könnten Sie), Futur I (plans), wenn (conditions) |
| **Travel - transport and connections** | Verbs with prepositions (warten auf, ankommen in), temporal clauses (wenn, bevor) |
| **Travel - problems and delays** | weil/da (cause), Konjunktiv II (polite complaints), Passiv (Der Zug wurde gestrichen) |
| **Accommodation - hotel** | Konjunktiv II (requests), Relativsätze (room description), damit/um...zu (purpose) |
| **Accommodation - problems** | weil (explanations), Konjunktiv II (polite complaints), Passiv (Das Zimmer wurde nicht gereinigt) |
| **Travel stories** | als (one-time events), Perfekt (narration), Relativsätze (place descriptions) |
| **Job hunting - CV** | Perfekt/Präteritum (experience), nachdem (sequence), Partizip II as adjective |
| **Job interview** | Konjunktiv II (politeness), weil/da (justifications), Futur I (career plans) |
| **At work** | damit/um...zu (purpose), Konjunktiv II (requests), Passiv (processes) |
| **Work - problems** | obwohl/trotzdem (contrast), weil (explanations), Konjunktiv II (suggestions) |
| **At the doctor - visit** | Konjunktiv II (polite questions), seit/seitdem (duration), wenn (conditional symptoms) |
| **Pharmacy - medicine** | Passiv (instructions), bevor/nachdem (medication sequence), Konjunktiv II |
| **Healthy lifestyle** | um...zu/damit (purpose), weil (justifications), Konjunktiv II (advice) |
| **Accidents and emergencies** | als (event description), Passiv (what happened), nachdem (sequence) |
| **Apartment hunting** | Relativsätze (apartment description), Konjunktiv II (questions), Genitiv (wegen der Lage) |
| **Furnishing** | Passiv (work), damit/um...zu (purpose), Partizip II as adjective |
| **Neighborhood** | obwohl/trotzdem (conflicts), weil (explanations), Konjunktiv II (requests) |
| **Shopping - complaints** | weil (problem), Konjunktiv II (polite demands), Passiv (what happened to product) |
| **Services** | Konjunktiv II (requests), damit/um...zu (purpose), während (during) |
| **Education** | nachdem (sequence), weil/da (justifications), Futur I (plans) |
| **Language learning** | obwohl (difficulties), um...zu (learning purpose), wenn (success conditions) |
| **Technology - problems** | Passiv (fault description), weil (cause), Konjunktiv II (help request) |
| **Internet and social media** | obwohl/trotzdem (pros and cons), weil (opinion justifications), Relativsätze |
| **Environment - ecology** | weil/da (causes), um...zu/damit (action purposes), Passiv (what is being done) |
| **Climate and weather** | wenn (conditions), Futur I (forecasts), weil (phenomenon causes) |
| **Family - relationships** | als (memories), obwohl (conflicts), Konjunktiv II (wishes) |
| **Friendship** | seitdem (since when), obwohl/trotzdem (problems), Konjunktiv II (apologies) |
| **Feelings and emotions** | weil (emotion causes), obwohl (contrast), Konjunktiv II (wishes) |
| **Apologizing and conflicts** | Konjunktiv II (if I had known), weil (explanations), obwohl (although) |
| **Society** | Passiv (what is being done), weil/da (justifications), um...zu (purposes) |
| **Politics** | Passiv (political processes), weil (justifications), Relativsätze (descriptions) |
| **Culture - entertainment** | Relativsätze (reviews), weil (opinions), als (event memories) |
| **Hobbies and free time** | seitdem (since when), weil (why I like), wenn (when I practice) |
| **Future plans** | Futur I, Konjunktiv II (dreams), wenn (conditions), damit/um...zu (purposes) |
| **Life turning points** | als/nachdem (sequence), weil (decision causes), Konjunktiv II (what if) |

## Differences Between A2 and B1

| Aspect | A2 | B1 |
|--------|----|----|
| Sentences | Simple, short | Complex, with conjunctions |
| Topics | Concrete, everyday | Abstract (opinions, plans) |
| Grammar | Basic | Konjunktiv II, Passiv, complex clauses |
| Vocabulary | ~1000 words | ~2500 words |
| Independence | Predictable situations | Unpredictable situations |
